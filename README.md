# Classification of Anomalies of Gastrointestinal Tract
- This repository contains code for training a classification model on KVASIR dataset of anomalies of gastrointestinal tract, using a deep convolution neural network with transfer learning
- Kvasir Dataset is a publicly released gastrointestinal tract image datset.In total, the dataset contains 8000 images that belongs to 8 different classes of anatomical landmarks and patholigical and normal findings.
- The dataset can be downloaded via : https://datasets.simula.no/kvasir
